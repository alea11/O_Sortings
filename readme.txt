решение состоит из 4 проектов:
1. Sortings - библиотека, набор различных алгоритмов сортироок, используется 3-мя остальными проектами решения.
2. SortConsole - консольное приложение , возможность выбора сортировки и задания параметров, - выводится информация об к-ве операций и времени выполнения.
	и только в этом приложении можно прогнать 2 сортировки с использованием файлов - ExternalMergeSort и RadixFileSort.
3. SortView - winforms приложение для визуализации процесса сортировки. при этом значительная часть времени тратится на отрисовку, поэтому в этом приложении сравнивать быстродействие именно сортировок - не стоит. 
4. Testing - консольное приложение автоматического тестирования сортировок. 
	для его работы предварительно нужно распаковать содержимое архивов (4 папки) в папку TestingFiles в корне решения.
